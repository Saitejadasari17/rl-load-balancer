🚀 HOW TO GET YOUR APP ONLINE (3 STEPS)
===============================================

STEP 1: Create GitHub Repository
---
git init
git add .
git commit -m "RL Load Balancer capstone"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/rl-load-balancer.git
git push -u origin main

STEP 2: Deploy Backend (Render.com)
---
1. Go to render.com
2. Click "New Web Service"
3. Connect GitHub repo
4. Build: pip install -r backend/requirements.txt
5. Start: cd backend && python -m uvicorn main:app --host 0.0.0.0 --port $PORT
6. Deploy!
7. Copy Render URL (e.g., https://rl-load-balancer-api.onrender.com)

STEP 3: Deploy Frontend (Vercel.com)
---
1. Go to vercel.com
2. Click "New Project" → Import GitHub repo
3. Framework: React
4. Root Directory: ./frontend
5. Add Environment Variable:
   REACT_APP_API_URL = [Your Render URL from Step 2]
6. Deploy!
7. Get Vercel URL (e.g., https://rl-load-balancer.vercel.app)
